Python

Python est un language de programmation avec une syntaxe simple et lisible. Il est utilisé par des grandes entreprises comme Google et même des agences gouvernementales comme la NASA.

Simplicité

Python gagne en popularité depuis 15 ans. C'est maintenant le premier language informatique utilisé pour l'initiation à la programmation dans les universités. En effet sa simplicité permets d'écrire moins de code et de se concentrer sur les concepts importants de la programmation.

Modules

L'installation de base de Python comprends un grand nombre de modules qui accélèrent le développement. Si tu ne trouves pas le module qu'il te faut parmis ceux qui sont installés par défaut, tu peux choisir parmis les 22 000 000 de modules installables avec pip.

Django

Le développement web est LE secteur important du moment. Effectivement le web évolue et de plus en plus d'applications sont développés pour être utiliser dans le navigateur. Django, un framework web en Python, est utilisé par des sites comme Pinterest pour sa simplicité et sa puissance.

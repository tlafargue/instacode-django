Outils de développement

Un bon développeur sait se servir des outils qui facilitent son travail et accélèrent le développement de ses applications.

Git

Le versionnement est un enjeu clé de la programmation. En tant que développeur, il est fréquent de devoir revenir à une version antérieure de son programme. La maîtrise de git est donc essentielle pour travailler efficacement en équipe.

GitHub

GitHub est la plus grande communauté open source au monde. Contribuer à des projets open source sur GitHub est la meilleure manière de prouver aux employeurs que l'on est capable de contribuer à un projet utile ainsi que d'écrire du code propre et lisible.

Trello

Trello est un outil collaboratif qui gagne en popularité chez les développeurs. Il permet de s'organiser avec plusieurs collaborateurs pour se répartir le travail. Utilisé en conjonction avec des méthodes d'organisation comme Scrum et Kanban, Trello est un outil essentiel au travail en équipe.
